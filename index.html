Welcome to our kitchen

This hasadna project supports open Primary election. The code contains a django project for a spcific party and allows the party members to ask and upvote questions and for candidates to answer.

There's a google drive collection that holds the design documents. You are invited to fork the code, improve the design and send a pull request.

Quick Start

You can access the live site or if you're a Django developer, install it on your local machine:

$ mkvirtualenv --no-site-packages okqa
$ git clone https://github.com/hasadna/ok-qa.git
$ cd ok-qa
$ cp okqa/local_settings_template.py okqa/local_settings.py
$ pip install -r requirements.txt
$ python manage.py syncdb --migrate
$ python manage.py runserver
